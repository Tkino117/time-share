# TimeShare

スケジュールを共有できる SNS アプリのバックエンド。
友達や他ユーザーの1日の過ごし方をチラ見できます。

## 主な機能

- **ホーム**: 自分・フォロー中ユーザーのスケジュール表示、イベントの追加・編集・削除
- **カレンダー**: 月間の予定管理
- **検索**: ID / 名前によるユーザー検索、フォロー、プロフィール閲覧
- **ランキング**: 睡眠・勉強など活動時間のランキング
- **通知**: フォロー / フォローリクエスト
- **設定**: プロフィール編集、鍵アカウント・非公開設定

## 技術構成

| 項目 | 採用技術 |
| --- | --- |
| 言語 | TypeScript |
| 実行環境 | Node.js |
| フレームワーク | Express |
| DB | SQLite (Sequelize) |
| プロキシ | nginx |
| サーバー | Sakura VPS |

## アーキテクチャ

テスト容易性のため、基本的に一方向依存のレイヤー構成を採用。

- `src/controller/` — レスポンス生成
- `src/service/` — 内部処理・ビジネスロジック
- `src/repository/` — データベース操作
- `src/database/` — モデル定義
- `src/router/` — ルーティング
- `src/response/` — レスポンス型定義

## セットアップ

```bash
npm install
npm start
```

サーバーは `http://localhost:3500` で起動します。

### テスト

```bash
npm test
```
